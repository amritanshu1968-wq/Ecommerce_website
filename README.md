# 📌 Overview

This project is an interactive **Ecommerce Sales Dashboard** developed using **Python**, **Streamlit**, **Pandas**, **Matplotlib**, **Seaborn**, and **Plotly**.

It helps businesses analyze their sales data through interactive charts and filters to gain meaningful insights for decision making.

✅ Interactive Dashboard

✅ Country-wise Filtering

✅ Revenue Analysis

✅ Monthly Revenue Trend

✅ Top 10 Countries by Revenue

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/amritanshu1968-wq/Ecommerce_website.git
```

Move into project folder

```bash
cd Ecommerce_website
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
